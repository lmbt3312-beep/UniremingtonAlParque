# UniremingtonAlParque
Descripción del proyecto

Uniremington Al Parque es una aplicación web desarrollada para la Corporación Universitaria Remington, cuyo objetivo es optimizar el registro, organización y seguimiento de las personas que participan en jornadas sociales realizadas en la comunidad.

La plataforma permite digitalizar la recolección de datos que tradicionalmente se realiza en planillas físicas, mejorando la trazabilidad de la información, facilitando el análisis de resultados y fortaleciendo el impacto social de las actividades institucionales.

🚀 Funcionalidades principales
Registro de usuarios con acceso restringido
Inicio de sesión seguro
Registro de asistentes en tiempo real
Clasificación por áreas (medicina, veterinaria y asesoría jurídica)
Almacenamiento estructurado en base de datos
Generación de reportes y estadísticas
Creación de recordatorios para seguimiento de casos
Funcionamiento offline con sincronización posterior
🎯 Objetivo

Brindar una herramienta tecnológica que permita mejorar la gestión de la información en las jornadas sociales, garantizando seguridad, organización y facilidad de acceso a los datos.

🔐 Seguridad

La aplicación está diseñada para manejar datos sensibles, implementando:

Control de acceso por roles
Validación de usuarios institucionales
Protección de datos personales
Manejo confidencial de la información
🛠️ Tecnologías utilizadas
MIT App Inventor
Figma
Canva
ClickUp (metodología Scrum)
Microsoft Office
📊 Valor del proyecto

Uniremington Al Parque no solo permite registrar información, sino también:

Analizar el impacto social
Mejorar la toma de decisiones
Realizar seguimiento a los beneficiarios
Optimizar los procesos institucionales
